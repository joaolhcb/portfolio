Definição das tecnologias:

HTML
O HTML, sigla para HyperText Markup Language, é uma linguagem de marcação que serve como base de uma página web. Isso quer dizer que com ela é permitida a construção de sites.
CSS
O CSS, sigla para Cascading Style Sheets, é um mecanismo utilizado para adicionar estilo a uma página web.
Javascript
JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma. Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web.

Prós e Contras:

Prós
Utilizar as três tecnologias juntas acaba tornando o processo mais fácil e possibilita que os projetos sejam apresentados em navegador. Temos uma complementação entre os três que ajuda na hora do trabalho.
Contras
Os projetos mais focados em jogos como o jogo da velha e o do dinossauro acabam ficando limitados apenas ao navegador. Se fosse utilizado em um software próprio talvez haveria mais facilidade na hora de projetar.
